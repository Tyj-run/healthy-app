📁 项目结构 (health-app)

health-app/
├─ src/
│ ├─ views/ // 页面视图组件
│ │ ├─ Dashboard.vue // 首页仪表盘
│ │ ├─ HeartRate.vue // 心率管理
│ │ ├─ SleepAnalysis.vue // 睡眠分析
│ │ ├─ Medicine.vue // 用药管理
│ │ ├─ Profile.vue // 个人档案
│ │ ├─ DataCharts.vue // 数据图表
│ │ └─ Login.vue // 登录页面
│ ├─ stores/ // Pinia 状态管理
│ │ ├─ auth.js // 登录认证
│ │ ├─ health.js // 健康指标
│ │ ├─ heartRate.js // 心率记录
│ │ ├─ sleep.js // 睡眠记录
│ │ ├─ medicine.js // 用药管理
│ │ └─ profile.js // 个人档案
│ ├─ router/ // Vue Router 路由
│ │ └─ index.js // 路由配置
│ ├─ layouts/ // 布局组件
│ └─ main.js // 应用入口
├─ index.html // HTML 入口
├─ vite.config.js // Vite 构建配置
├─ package.json // 依赖管理
└─ README.md // 项目说明

技术栈

Vue 3 + Vite 5 + Pinia + Vue Router + Ant Design Vue + ECharts

采用 Composition API 风格，状态管理使用 Pinia 并配合 localStorage 持久化

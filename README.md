# 健康管家（Health Manager）

一款基于 Vue 3 的移动端优先健康数据管理应用，集成用药提醒、健康指标监测、数据可视化等功能，帮助用户日常追踪和管理个人健康状态。

---

## 技术栈

- **框架**：Vue 3（Composition API）
- **构建工具**：Vite 5
- **UI 组件库**：Ant Design Vue 4
- **状态管理**：Pinia 2
- **路由**：Vue Router 4
- **图表库**：ECharts 5 + vue-echarts
- **语言**：JavaScript
- **图标**：Font Awesome 6

---

## 功能特性

### 登录页面

- 用户名密码登录验证
- 登录状态持久化（localStorage）
- 3 分钟登录有效期，动态倒计时，到期自动提示重新登录
- 路由守卫拦截未授权访问，注销后手动输入地址也强制跳转登录页

### 首页仪表盘

- 用户欢迎信息动态展示
- 快捷功能入口：药品识别、用药提醒、设备管理、健康报告
- 今日用药提醒列表，实时同步药品页数据
- 健康数据卡片：心率、血糖、睡眠、步数，点击跳转详情
- 本周血糖趋势图（ECharts 折线图）

### 药品管理页

- **条件筛选**：支持按药品名称关键词实时搜索过滤
- **添加药品**：
  - 常见药品库选择（含儿童/成人剂量智能提示）
  - 手动输入药品信息
  - 剂型选择（片剂、胶囊、冲剂、口服液、滴剂、喷雾）
  - 每次用量调节（0.25 步进）
- **编辑药品**：修改用药时间、剂量、重复周期、提醒开关
- **删除药品**：确认后移除，数据同步更新
- 用药提醒设置：多时间配置、重复周期（周一至周日）、语音提醒开关
- 全部数据 localStorage 持久化，刷新不丢失

### 数据图表页

- 实时心率展示（Canvas 动画圆环）
- 心率趋势图（过去 7 天，自动读取系统日期）
- 睡眠时长柱状图（过去 7 天）
- 本周健康报告：BMI 动态计算（依据个人信息身高体重）、健康评分、用药完成率
- BMI 说明弹窗：公式展示、分类标准（偏瘦/正常/超重/肥胖）

### 个人中心页

- 头像上传与更换（本地相册选择 + Canvas 圆形裁剪）
- 姓名编辑并持久化保存
- 个人信息展示：性别、年龄、身高、体重、血型、邮箱、手机号
- 编辑个人信息弹窗，修改后自动同步 BMI 计算
- 设置入口：账号管理、深色模式、通知管理、数据同步、清除缓存
- 帮助中心：使用指南、常见问题、隐私政策、关于我们

### 账号管理

- 查看/修改绑定邮箱、手机号
- 修改密码（留空则不修改）
- 动态显示登录剩余时间倒计时
- 保存账号信息
- 退出登录（清除 token、登录态，保留个人资料）

### 通知与提醒系统

- 顶部通知面板，消息列表展示
- 通知总开关：关闭时一键关闭所有提醒，开启时恢复之前的提醒设置
- 用药闹钟：定时检查 + 语音播报（Web Audio API 合成提示音）
- 弹窗提醒，不突兀退出

### 深色模式

- 一键切换深色/浅色主题
- CSS 自定义属性全局覆盖
- 主题偏好 localStorage 持久保存

## 登录说明

- 测试账号：`admin` / 任意密码
- 登录后有效期为 **3 分钟**，过期后需重新登录
- 所有登录态存储在 `localStorage` 中，刷新页面不会丢失
- 个人资料（姓名、头像、身高体重等）独立于登录态，换账号后依然保留

---

## 数据持久化说明

| 数据类型 | localStorage 键 | 说明 |
|---------|----------------|------|
| 登录态 | `token`, `user`, `expiresAt` | 注销时清除 |
| 个人资料 | `health_profile` | 长期保留 |
| 头像 | `avatarUrl` | 长期保留 |
| 药品列表 | `health_medicines` | 长期保留 |
| 健康指标 | `health_metrics` | 长期保留 |
| 主题偏好 | `theme` | 长期保留 |

## 核心实现亮点

- **路由守卫**：直接从 `localStorage` 读取 token 与过期时间判断登录态，不依赖 Pinia 内存状态，防止刷新后守卫失效。
- **动态倒计时**：使用 `ref` + `setInterval` 每秒读取 `localStorage` 中的 `expiresAt`，实现真正的实时倒计时，而非缓存的 `computed`。
- **Canvas 圆形头像**：上传图片后通过 `canvas.clip()` 裁剪为圆形，并导出为 DataURL 持久化。
- **Web Audio API**：闹钟触发时合成提示音并语音播报（使用 `SpeechSynthesisUtterance`），无需外部音频文件。
- **数据同步**：首页健康卡片、步数、用药提醒均通过 Pinia store 的 `computed` 实时联动，所有变更自动写入 `localStorage`。
